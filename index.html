R-CAEn

My current personal project, R-CAEn (pronounced 'Arcane'), is a cellular-automata-based game engine that greatly facilitates the creation and simulation of cellular automata, as well as possessing many other features such as layer-to-layer communication and greatly improved performance thanks to the CBPO (Chunk-Based Performance Optimization) algorithm I developed.

Source code is not publicly available but I'm willing to provide snippets for interview purposes.

  • Download Demo

Chunk-Based Performance Optimization Algorithm

While developing R-CAEn, I noticed a steep performace drop as a result of all the function calls, so in my quest for optimization, I developed the "Chunk-Based Performance Optimization Algorithm For Two-Dimensional Square Lattice Cellular Automata", which in essence, marks the active parts of the grid to be iterated over, instead of iterating over every single cell, every single step, greatly improving performance in most cases.

Montrez Vos Muscles!

This is the project I worked on towards the end of my internship at Digital Mania Studio, it is my first ever professional work experience.

It is a game that uses Intel's RealSense™ motion-tracking technology, and Nuitrack's skeletal tracking SDK to allow the player to move an avatar in real time.
I worked on the initial tracking system, the player's X-ray shader, the wall texture shader, the score system, and the UI functionality & animation.

The client ended up abandoning the project when they realized the idea was too ambitious.

Unity 2D Shooter AI Prototype

I developed this as a tech demo for DigitalMania Studio.

This system is comprised of a number of triggers for both walking and jumping, connected together through an A* algorithm which determines which triggers are to be activated for the AI to reach it's destination, as well as a linecast function to determine if the AI has an unbroken line of sight to the player in order to shoot.
